## Описание сервиса Dashboard

Сервис Dashboard предназначен для удобного отображения списка работников с назначенными на них заявками и соответствующей информацией из ERP Cubalink (Userside). Некоторые элементы блока содержат ссылки на ERP-сервис с примененными фильтрами.

**Информация о блоке работника**

Блок работника включает в себя следующие данные:

- Имя работника
- Запись в табеле (На работе / выходной / не записан)
- Количество отработанных дней в текущем месяце (ссылка)
- Количество выходных дней в текущем месяце (ссылка)
- Количество назначенных на мастера заявок (ссылка)
- Количество заявок в исполнении (в идеале не больше одной) (ссылка)
- Количество выполненных заявок за текущий день (ссылка)
- Количество выполненных заявок за текущий месяц (ссылка)
- Дата и время последней выполненной заявки

Если у работника есть заявки в исполнении, информация по ним отображается ниже в отдельном блоке внутри блока работника.

Цветовая схема заявок указывает на то, прошло ли 3 часа после назначения её работнику в исполнение:
- <font color="green">Зеленый цвет</font> : прошло менее 3 часов.
- <font color="red">Красный цвет</font> : прошло более 3 часов.

**Детальная информация о заявке**

Для каждой заявки в исполнении отображается следующая информация:
- Номер заявки (ссылка)
- Время, прошедшее после назначения заявки в исполнение
- Дата и время назначения работнику заявки в исполнение
- Тип заявки и подкатегория заявки
- Логин, ФИО и Адрес из полей, заполненных оператором и полей из Билинга.<br>
