**Dashboard** - сервис для удобного отображения списка работников с назначенными на них заявками и информации по ним из ERP Cubalink (Userside). Некоторые элементы блока, ссылаются в ERP-сервис с выставленными фильтрами.

Блок работника включает в себя следующую информацию:
- Имя работника
- Запись в табеле (На работе / выходной / не записан)
- Количество отработанных дней в текущем месяце (url)
- Количество выходных дней в текущем месяце (url)
- Количество назначенных на мастера заявок (url)
- Количество заявок в исполнении (в идеале не больше одной, но они не идеальны) (url)
- Количество выполненных заявок за текущий день (url)
- Количество выполненных заявок за текущий месяц (url)
- Дата и время последней выполненой заявки

Если у работника есть заявки в исполнении, информация по ним отображается ниже, отдельным блоком, внутри блока работника.
- Цветовая схема заявки показывает, прошло ли 3 часа после назначения её работнику в исполнении - красный цвет. Если нет - зеленый цвет.
- Номер заявки (url)
- Сколько времени прошло, после назначения заявки в исполнении
- Дата и время назначения работнику заявки в исполнении
- Тип заявки. Подкатегория заявки
- Логин, ФИО и Адрес из полей заполненных оператором и полей из Билинга
